## [0.1.0] - 2025-01-XX

### Added
- Initial release
- Integration with OpenAI, Claude (Anthropic), and Google Gemini AI providers
- Database schema loading from SQL files and YAML
- Rules and conventions configuration via YAML files
- SQL query generation from natural language input
- SQL validation to prevent dangerous operations
- Rails integration with optional controller and routes
- Customizable prompts and AI parameters
- Comprehensive test suite
- RuboCop linting configuration
